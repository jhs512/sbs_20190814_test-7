■ 테스트 목록
- 공지사항은 관리자만 작성할 수 있다.
- 오직 회원만 글을 작성할 수 있다.
- 게시물/댓글의 삭제는 관리자나 작성자만 가능하다.
- 게시물/댓글의 수정은 작성자만 가능하다.
- 회원가입 후 이메일 인증까지 되어야 로그인이 가능하다.
- 비밀번호 찾기는 회원의 이메일로 임시패스워드를 발급하는 과정으로 진행되어야 한다.
- 탈퇴한 회원의 정보로는 IP/PW 찾기를 이용할 수 없어야 한다.
- 탈퇴한 회원의 ID나 이메일 정보로는 가입 할 수 없어야 한다.
- 기타 단순한 오류들 체크
	- 리스트에 작성자 없음
	- 게시글 상세페이지에 작성자 없음
	- 댓글에 작성자 없음

■ 테스트 보고서
□ 공지사항은 관리자만 작성할 수 있다.
- 오류내역
	●오류 1
	●내용 : 정상적인 UI 경로를 타지 않고 직접 URL 입력으로 접근하면 관리자가 아닌 회원도 공지사항 글쓰기가 가능하다.
	●관련정보
	URL : http://localhost:8083/article/add?boardId=1
	●추가 오류 1-1
	URL : http://localhost:8083/article/doAdd?boardId=1&title=a&body=b

□ 오직 회원만 글을 작성할 수 있다.
- 오류내역
	●없음

□ 게시물/댓글의 삭제는 관리자나 작성자만 가능하다.
- 오류내역
	●오류 2
	●내용 : 정상적인 UI 경로를 타지 않고 직접 URL 입력으로 접근하면 관리자나 작성자가 아니어도 삭제가 가능하다.
	●관련정보
	(글 삭제)http://localhost:8083/article/doDelete?id=5&boardId=2
	●추가 오류 2-1
	●내용 : 관리자도 게시물/댓글을 삭제할 수 있어야 한다.

□ 게시물/댓글의 수정은 작성자만 가능하다.
- 오류내역
	●오류 3
	●내용 : 정상적인 UI 경로를 타지 않고 직접 URL 입력으로 접근하면 관리자나 작성자가 아니어도 삭제가 가능하다.
	●관련정보
	댓글 수정 URL : http://localhost:8083/article/doModifyReply?id=1&body=아무글
	글 수정 URL : http://localhost:8083/article/modify?id=7&boardId=2
	글 수정 URL : http://localhost:8083/article/doModify?title=10&body=10&boardId=2&id=7

□ 회원가입 후 이메일 인증까지 되어야 로그인이 가능하다.
- 오류내역
	●없음

□ 비밀번호 찾기는 회원의 이메일로 임시패스워드를 발급하는 과정으로 진행되어야 한다.
- 오류내역
	●오류 4
	●내용 : 임시패스워드가 아닌 회원이 실제 비밀번호를 발급한다.
	●관련정보
	길이가 5인 랜덤키 생성하여 DB 업로드 후 발급하는 과정으로 변경

□ 탈퇴한 회원의 정보로는 IP/PW 찾기를 이용할 수 없어야 한다.
- 오류내역
	●오류 5
	●내용 : 탈퇴한 회원도 정보를 입력하면 이메일로 정보가 발급된다.

□ 탈퇴한 회원의 ID나 이메일 정보로는 가입 할 수 없어야 한다.
- 오류내역
	●없음

□ 기타 단순한 오류들 체크
- 리스트에 작성자 없음
	●오류 6
- 게시글 상세페이지 글/댓글에 작성자 없음
	●오류 7
